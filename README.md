# CS 330 – Computational Graphics and Visualization  

08/24/25
Jessica Johnson

This repo holds my CS 330 final project: a 3D makeup vanity scene built in OpenGL. I created objects like lip gloss, a compact, a nail polish bottle, and brushes using a mix of cylinders, cones, and spheres. This was my first time really seeing my code come alive visually, and it gave me a whole new appreciation for the graphics side of programming.  

## What’s in this repo
- Source code (`.cpp` and `.h` files)  
- Shaders and textures  
- Design Decisions document (PDF/DOCX)  
- Reflection answers (below)  
---

## Reflection  

### 1) How do I approach designing software?  
I start simple and break things down into basic shapes or pieces I can build on. For this scene, that meant sketching out which objects would use spheres, cylinders, or cones before worrying about details. From there, I used textures and lighting to add realism. The build it "step by step” approach kept me on track without getting overwhelmed.  

### 2) How do I approach developing programs?  
I like to test small changes as I go instead of waiting until the end. For this project, I added one object at a time, tested it, and then layered on textures and lighting. Milestones helped me iterate in stages as well. Geometry first, then interaction, then textures and light. That made it easier to fix issues along the way.  

### 3) How can computer science help me in reaching my goals?  
Even though I focus more on backend development, this course showed me how powerful visualizations can be. I now understand more about how lighting, textures, and shaders bring a scene to life. These skills give me another toolset for future school projects and a portfolio piece I can show off. It’s also motivating to know I can take on graphics if I ever need to in my career.
